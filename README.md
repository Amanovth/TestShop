# TestShop

## Installation

1. Clone the repository using HTTPS:
```shell
git clone https://github.com/Amanovth/TestShop.git

```
Or using SSH:

```shell
git clone git@github.com:Amanovth/TestShop.git

```

2. Go to repository:
```shell
cd TestShop/
```

3. Run the container:
```shell
docker-compose up --build
```
